### Zahlen

* `int` (Ganzzahlen)
* `float` (Kommazahlen)
* Grunderechenarten `+`, `-`, `*`, `/`, `//`
* Potenz `**`
* Modulo `%`
* `float` -> `int` (möglicher) Informationsverlust
* `int` -> `float` kein Informationsverlust
* `float` sind etwas komplizierter als `int`

```py
>>> 0.1 + 0.1 + 0.1
?
```
